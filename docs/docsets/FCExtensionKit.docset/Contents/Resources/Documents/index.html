Installation

Swift Package Manager

dependencies: [
    .package(url: "https://github.com/fatihcimen/FCExtensionKit", .upToNextMajor(from: "1.1.0"))
]

CocoaPods

Add the following to your Podfile:

pod 'FCExtensionKit', '~> 1.1.0'
